Retail Sales Analysis Project

Introduction:
* This project analyzes retail sales data to identify business trends and generate actionable insights using:
   1.Excel
   2.Python
   3.Tableau. 
* It demonstrates an end-to-end data analysis workflow, from data exploration and validation to dashboard creation.

Objective:
* Analyze sales performance across different regions.
* Identify monthly sales trends.
* Compare sales across product categories.
* Discover top-performing products.
* Present insights using interactive visualizations.

Dataset:
The dataset contains retail sales information, including:
* Row ID
* Order ID
* Order Date
* Ship Date
* Ship Mode
* Customer Details(ID, Name, Country, City, State, Postal Code)
* Segment
* Region
* Category & Sub-Category
* Product ID
* Product Name
* Sales

Tools & Technologies:
1.Microsoft Excel:
  * Pivot Tables
  * Charts
  * Data Analysis

2.Python:
  * Pandas
  * Matplotlib

3.Tableau Public:
  * Interactive Dashboard
  * Data Visualization

Project Workflow:
1. Data Preparation
* Loaded and explored the retail sales dataset.
* Verified data structure and performed basic validation.

2. Excel Analysis
* Created Pivot Tables and Charts for:
	* Sales by Region
	* Monthly Sales Trend
	* Sales by Category
	* Product Sales Ranking

 3. Python Analysis
* Performed:
	* Dataset loading using Pandas
	* Sales summary calculations
	* Regional sales analysis
	* Monthly sales trend analysis
	* Top-selling product identification
	* Business insight generation

4. Tableau Dashboard
* Built an interactive dashboard containing:
	* Sales by Region
	* Monthly Sales Trend
	* Sales by Category

Key Insights:
	* Identified the highest-performing sales region.
	* Analyzed monthly sales trends to understand seasonal patterns.
	* Compared sales performance across product categories.
	* Identified top-performing products using Python analysis.
